# CI/CD Report 0017

Data: 2026-03-03
Escopo: automacao completa de CI e CD com Supabase + Vercel.

## Workflows criados

- `/.github/workflows/ci.yml`
  - triggers: `pull_request`, `push`
  - etapas:
    - checkout
    - setup node 20
    - `npm ci`
    - `npm run verify`

- `/.github/workflows/cd.yml`
  - trigger: `push` na branch `main`
  - etapas:
    - checkout
    - setup node 20
    - validacao explicita de secrets obrigatorios
    - `npm ci`
    - `npm run verify`
    - setup supabase cli
    - `supabase login` + `supabase link`
    - `npm run db:deploy`
    - `vercel deploy --prod --yes`
